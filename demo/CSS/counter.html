计数器

counter 的基本使用(最常用)

        .counter1{
            counter-reset: count1;
        }
        .counter1 li::before {
            counter-increment: count1;
            content: counter(count1) ": ";
        }

counter 的进阶使用

        .counter2 {
            counter-reset: count2;
        }

        .counter2 li::before {
            counter-increment: count2 2;
            content: counter(count2) ": ";
        }

        .counter3 dl li::before {
            counter-increment: count4;
            content: "二级目录："counter(count3) ": "counter(count4, lower-roman);
