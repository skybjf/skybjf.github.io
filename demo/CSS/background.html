关于 linear-gradient 的使用

linear-gradient 的基本使用

        #grad1 {
            height: 100px;
            background-image: linear-gradient(90deg, green , yellow );
        }

linear-gradient 的进阶使用

        #grad2 {
            height: 100px;
            background-image: linear-gradient(90deg, green 50%, yellow 50%);
            background-size: 30px 100%; 
        }
        
        #grad3 {
            height: 100px;
            background-image: linear-gradient(180deg, green 30%, yellow 30%, yellow 60%,blue 60%);
            background-size: 100% 30px; 
        }

linear-gradient 的演变使用

        #grad4 {
            height: 400px;
            background-image:
                linear-gradient(45deg, #eee 25%, transparent 25%, transparent 75%, #eee 75%),
                linear-gradient(45deg, #eee 25%, transparent 25%, transparent 75%, #eee 75%);
            background-size: 16px 16px;
            background-position: 0 0, 8px 8px;
        }
