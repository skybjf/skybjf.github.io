### npm指令

* 安装模块：npm install xxx
* 安装模块到全局：npm install xxx -g
* 查看已安装的模块：npm ls
* 查看已安装到全局的模块：npm ls -g
* 卸载模块：npm uninstall xxx (-g)
* 清理缓存：npm cache clean
